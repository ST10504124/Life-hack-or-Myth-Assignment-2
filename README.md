## Background
 
Life Hack or Urban Myth is an interactive flashcard app designed to help users distinguish between genuine productivity tips and misleading internet myths. With so many viral “life hacks” online, it can be difficult to tell what is actually useful or safe.
The app addresses this problem by presenting users with a series of flashcards where they must decide whether each tip is a real life hack or an urban myth. Users receive feedback and a final score based on their answers, helping them learn and improve their judgment.
Overall, the app provides a fun and educational way to test common sense, promote critical thinking, and encourage safer, more practical everyday habits.

## Getting Started (Installation)
 
### Prerequisites
 
- Android Studio Jellyfish (or latest version) - Kotlin SDK - JDK 17 (or compatible version) - Git installed on your local machine
 
### Installation Steps
 
1. **Clone the repository:**
 
       git clone [https://github.com/ST10504124/Life-hack-or-Myth-Assignment-2] 
 
2. **Open the project:** Launch Android Studio and select `Open`, then navigate to the cloned folder.
 
3. **Sync Project:** Wait for the Gradle sync to complete.
 
4. **Run the App:** Connect an Android device or start an emulator and click the **Run** button (Green Play Icon).
